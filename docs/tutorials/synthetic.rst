Step-by-step synthetic example
==============================

In this tutorial, we walk through step-by-step the process used to create the synthetic examples as shown in the QuakeMigrate accompaniment paper. The aim of this is to provide some motivation for the various parameters that need to be tuned for a given QuakeMigrate run.
